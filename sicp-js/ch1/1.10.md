### Exercise 1.10
- `A(1,10) = 1024`  
- `A(2,4) = 65,536`  
- `A(3,3) = 65,536`  

- $f(n) = 2n$  
- $g(n) = 2^n$  
- $h(n) = 2^{2\dots 2}$  

```js
import {display} from 'sicp';

display(A(1,10));
display(A(2,4));
display(A(3,3));

function A(x, y) {
    return y === 0
           ? 0
           : x === 0
           ? 2 * y
           : y === 1
           ? 2
           : A(x - 1, A(x, y - 1));
}
```
