| Stage      | cmovXX rA, rB              |
| ---------- | -------------------------- |
| Fetch      | icode:ifun <- M1[PC]       |
|            | rA:rB <- M1[PC+1]          |
|            | valP <- PC + 2             |
| Decode     | valA <- R[rA]              |
| Execute    | valE <- 0 + valA           |
|            | Cnd <- Cond(CC,ifun)       |
| Memory     |                            |
| Write Back | if (Cnd) R[rB] <- valE     |
| PC Update  | PC <- valP                 |
