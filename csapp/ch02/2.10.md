### Exercise 2.10
If we set x = 4 and y = 5, the table would look like this  
| Step      | `*x`                      | `*y`                    |
| --------- | ------------------------- | ----------------------- |
| Initially | `a`                       | `b`                     |
| Step 1    | `a`                       | `a^b`                   |
| Step 2    | `a^(a^b) = (a^a)^b = b`   | `a^b`                   |
| Step 3    | `b`                       | `b^(a^b) = (b^b)^a = a` |
